registerDrawableObjectType

static registerDrawableObjectType({cls})
cls
Class object representing the object type.
Any user-defined model classes need to be registered so that the class objects can be serialized. The name will used as the class identifier in the serialized data. Example: 

class MyModelingModel extends Xc3dDocDrawableObject {
  ...
}

Xc3dDocDocument.registerDrawableObjectType({
  cls: MyModelingModel
});
