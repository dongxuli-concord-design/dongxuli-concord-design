How to create an app

  1. Create a new folder in Apps following the template of Hello.
  2. Modify the code.
  3. Copy the compiled binary code or source code, including resource files, to XcMain.
  4. Modify XcSysConfig.apps array in the config.js in the XcMain to load the app.
  5. Check Hello App for an example.

An example of config.js is shown in the following:

        
            XcSysConfig.title = '协和 Concord';
            XcSysConfig.server = 'http://concord.design';
            XcSysConfig.locale = 'zh';
            XcSysConfig.debug = true;
            XcSysConfig.apps = [
            'Apps/Foo1', // This will load 'Foo1.bin' (if available) or 'Foo1.js' in 'XcMain/Apps/' folder.
            'file:///YourFileFolder/Foo2', // This will load 'Foo2.bin' (if available) or 'Foo2.js' in '/YourFileFolder/'' folder.
            ];
